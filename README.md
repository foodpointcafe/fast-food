# HALAL Food Point - React Website

Сайт, разработанный на реакте для HALAL Food Point с дизайном, точно воспроизводящим стилистику меню из PDF. Сайт работает по ссылке: https://malikoshhh.github.io/foodpoint/

# Технологии

- **React 18** - UI библиотека
- **Vite 5** - сборщик и dev-сервер
- **CSS3** - стилизация (градиенты, flexbox, grid, анимации)
- **Google Fonts** - Bebas Neue, Montserrat

## Структура проекта

```
foodpoint/
├── index.html              # HTML точка входа
├── package.json            # Зависимости проекта
├── vite.config.js          # Конфигурация Vite
├── .gitignore              # Игнорируемые файлы
├── README.md               # Документация
└── src/
    ├── main.jsx            # Точка входа React
    ├── App.jsx             # Главный компонент
    ├── App.css             # Стили компонента
    └── index.css           # Глобальные стили
```

### Что реализовано:
- Интерактивные кнопки с эффектом skew
- Плавная смена категорий
- Активное состояние для текущей категории
- Градиентный фон
- Анимация при наведении
- Цены в красных бейджах
- Подробное описание состава
- Информация о доставке
- Логотип быка в круге
- Звёздный рейтинг
- Слоган бренда
